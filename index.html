Identification of VALUE

We identify in collaboration with our business partners their understanding about what value means, and to whom.

The outcome is an explicit and common vocabulary, so taking away subjectivity and misunderstandings.

Decision Support for VALUE

Each stakeholder assesses the impact that the implementation of a particular set of decisions will have from the identified VALUE factors.

The VALUE Tool provides a dashboard to provide different visualizations to guide discussions and decisions.

Prediction of VALUE

Our next goals are to identify a semi-automatic method for uncertainty quantification in Bayesian Network and to implement and evaluate the second generation tool that enables the use, customisation and building of Bayesian Network models.
